Part 2

The command below is used to create a new session with the ID # 001.

mySession = new App.Session('001', new App.TSM());

Create a new message with the command below:

mySession.createMessage({emailAddress: 'insert unique email here', msg:'insert message here'});

To mark a question that has been answered, use the command below:

mySession.answerMessage('insert email/sender ID here');

To check on the remaining unanswered messages, use the following command:

mySession.allMessages();

Part 1

My idea is that fans can send in questions (or any message) for the team.

To start off, you'll need the command below to create a new object.

var msg = new App.TSM();

To add a question, user has to include their name (doesn't have to be real name) and the question/message they want to send to the team, by using this command.

msg.add('Name', 'Message/Question');

User can look up a question by search for sender's name.

msg.get('Name');

User can also pull of all the questions available by this command.

msg.getAll();

A question can be deleted via selecting a user's name.

msg.remove('Name');
